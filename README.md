# Folder Manager

## Installation

### Clone git repository

```bash
git clone https://github.com/b-l-i-n-d/folder-manager.git
cd folder-manager
```

### Install dependencies

```bash
npm install
```

### Run the app

```bash
npm run dev
```

-   [x] Explore folder
